# Locally Weighted Regression (LWR)

This project implements Locally Weighted Regression (LWR), a non-parametric regression technique, from scratch using Python.

## Features
- Gaussian kernel-based weighting
- Weighted least squares solution
- Prediction for query points
- Visualization for different bandwidth values (tau)

## Dataset
Vehicle Speed vs Engine Torque dataset (vehicle_speed_torque.csv)

## Output
- Regression curves for different tau values (5, 20, 50)
- Visualization of model fit

## Tools Used
- Python
- NumPy
- Pandas
- Matplotlib

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Open the notebook:
   lwr_regression.ipynb

## Concepts Covered
- Locally Weighted Regression (LWR)
- Non-parametric models
- Kernel methods
- Bias-variance tradeoff
